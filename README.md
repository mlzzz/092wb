# 092五笔

## 简介:
>092输入法即09输入法第二代，由「行走的风景」发明，以其惊人的词组离散能力和高速输入成绩，震惊跟打界。可惜因风景、狂练等老一辈高手名宿的退隐，〇九大法又略显式微。后又有小友、凌风等年轻高手脱颖而出，再现中兴之象。
